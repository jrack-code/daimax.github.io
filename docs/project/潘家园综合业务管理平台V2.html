JK 2019-04-01

介绍

潘家园综合业务管理系统2.0主要解决，市场部各个摊位的租赁合同的批量录入、续签、终止，同时也为财务部提供收费、银行划账、预算等功能

# 录入合同

通过合同录入、合同批量导入的合同

# 续签合同

通过合同续签功能续签成功的合同

# 终止合同

已终止，且排除作废的合同
租赁合同中全是排除虚拟合同的数据，石雕区除外

# 摊位预算管理

预算生成方式
先向预算Biz_Cbd_Budget_Details表添加当年或明年预算数据，然后登录系统，在摊位预算管理模块，上半年数据中，右侧的操作按钮，点击生成功能

# 合同签约情况占比分析

续签

  • 查询时间范围内，非续签且不是作废的合同

新入市

  • 查询时间范围内，未终止的，并且原合同id为空的合同（当前逻辑-错误）
  • 查询时间范围内，所有原合同id为空并且终止状态为非作废的合同（正确逻辑）

退市

  • 查询时间范围内，终止的，没有续签的合同

*注意：所有逻辑包含虚拟合同
