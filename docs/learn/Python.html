# pip 快速安装Python项目依赖包

requirements.txt 用来记录项目所有的依赖包和版本号，只需要一个简单的 pip 命令就能完成。

pip freeze > requirements.txt

执行后会在项目目录生成requirements.txt，内容大概如下

alembic==1.4.2
altgraph==0.16.1
asgiref==3.2.9
astroid==2.3.3
autopep8==1.5.1
baidu-aip==2.2.18.0
beautifulsoup4==4.9.3
1
2
3
4
5
6
7

然后执行下面语句，进行安装

pip install -r requirements.txt
